Welcome to the official repository for the **HoloTSH Ecosystem**. This project provides a mathematically provable, neuro-symbolic geometric guardrail designed to prevent hallucinations and ensure structural safety in Large Language Models (LLMs) and Embodied AI agents.

## 🔬 1. NMI Paper Reproducibility

For reviewers and researchers looking to reproduce the findings from our manuscript submitted to *Nature Machine Intelligence* (e.g., the 0.82 ms latency and 97.66% signal recovery rate), please navigate to the `NMI_Reproducibility/` directory.

- Run `python OneButtonFire.py` for automated data fetching and manifold alignment benchmarking.

## 🛡️ 2. HoloAuditor: The "Antivirus Engine" for Medical LLMs

HoloAuditor applies our topological circuit breaker to clinical AI. We recognize that building a comprehensive medical mind is an ongoing journey. Therefore, HoloAuditor is designed with an **Engine + Signature** architecture:

- **The Core Engine:** The mathematical tensor-Laplacian filter (providing robust topological bounding).
- **The Signature Database:** An open, community-driven database of verified medical mappings (e.g., Symptom $\to$ Basic Syndromes). 
- **Contribute:** We invite clinicians and researchers to enhance our initial "V1.0 Knowledge Base" by submitting new clinical signatures via Pull Requests. Please see `HoloAuditor/CONTRIBUTING.md` for our peer-review pipeline.

## 🛑 3. HoloBlocker: Embodied AI Safety

Extending beyond text, the `HoloBlocker/` module contains the implementation for real-time topological interception in physical environments (e.g., surgical robots, autonomous drones), as detailed in our SSRN preprint (ID: 6421619).
